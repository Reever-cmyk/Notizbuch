# Notizbuch
java Notizbuch programm

Notizbücher können angelegt bearbeitet und erstellt werden. In den Notizbüchern gibt es die Möglichkeit neue Notizen zu erstellen, löschen oder zu bearbeiten.

erstellt man ein Notizbuch erstellt java ein Ordner in welchem die Notizen als .txt Dateien gespeichert werden.
